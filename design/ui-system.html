One spine, two registers.

A shared foundation keeps it one product; each surface gets a register tuned to its job. Web is a precise productivity workbench (Linear/Notion). Mobile is characterful and rewarding (Arc — sophisticated, not cartoon). The thread between them: one blue, one mastery-state language, one set of type roles.

01

Approach

Same spine, two visual registers — divergent feel, shared identity.

Web register — precise productivity

the management workbench · Linear / Notion
  • Near-grayscale slate + one functional blue accent
  • High density — three-pane, compact rows, filter chips
  • 14px base, keyboard-first (⌘K · 1–4 · space · enter)
  • Fast, functional motion · no celebration

Mobile register — characterful & rewarding

the daily ritual · Arc
  • Amplified blue brand + amber celebration, warm neutrals
  • Comfortable single column, ≥44px targets, 16px base
  • Instrument Serif for hero moments · thumb-first · haptics
  • Expressive motion — only at moments of progress
Shared spine

Gulp Blue · the 5-state mastery language · Inter + Geist Mono + Instrument Serif roles · the Card as atom · 4px base · plain voice.

Blue means “act”

The brand blue is rationed to the one key action per screen — and to Due, which is a call to act.

State is always legible

Every knowledge object shows its mastery state without a tap — color and label, never color alone.
